Glosario De Etiquetas

br: Inserta un salto de línea dentro de un párrafo.
hr: Inserta una línea horizontal para separar contenido.
strong: Resalta un texto en negrita para indicar importancia.
em: Aplica cursiva para dar énfasis a una parte del texto.
blockquote: Define una cita extensa con una fuente externa opcional.
code: Muestra un fragmento de código en estilo monoespaciado.
pre: Muestra un bloque de código preformateado.
sup: Superíndice para mostrar texto elevado, útil para fórmulas matemáticas.
sub: Subíndice para mostrar texto reducido, como fórmulas químicas.
table: Crea una tabla con encabezados (th) y celdas de datos (td).
img: Muestra una imagen con atributos para la fuente, dimensiones, título y estilo.
audio: Inserta un reproductor de audio con controles.
